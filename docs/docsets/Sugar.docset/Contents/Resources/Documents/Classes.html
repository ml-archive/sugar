  • JWTAuthenticationMiddleware

    A middleware that authenticates by extracting and verifying a JWT (JSON Web Token) from the Authorization header. By default it will also load the user (or whatever authenticatable entity is used) from the database but this can be disabled for performance reasons if it is not needed.

    Authentication is skipped if authentication has already occurred, e.g. in a previous middleware.
